# 🎰 SCAI Lucky Loop

A decentralized Web3 Lottery Platform built during the EtherAuthority Internship Program.

## 🚀 Features

- 🔐 MetaMask Wallet Integration
- 🎟️ Buy Lottery Tickets
- 💰 Live Prize Pool
- 👥 Referral Program
- 🏆 Previous Winner Display
- ⛓️ Secure Chain AI Smart Contracts
- 📱 Telegram Web App Support
- 🌐 Responsive React Frontend

## 🛠️ Tech Stack

- React.js
- Vite
- Ethers.js
- Solidity
- Hardhat
- OpenZeppelin
- Secure Chain AI Network
- Telegram Web App
- Vercel
